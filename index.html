    • The Color Game

      Description

      A color guessing game that takes Red,Green,Blue each color goes from 0 (non existent) to 255(highest), mixing these numbers gives a certain color that you have to guess by just looking at the numbers you can Reset and Play again and levels of difficulty.

      Technical stuff

      using vanilla JavaScript that generates random colors from 0 to 255 for each color, then runs this function that manipulating the DOM for six Divs, giving them different color, only one matches the correct answer, then it changes the background color to the right answer when picked.

    • TalkToMe

      Description

      “TalkToMe is a social network where you can create an account, password reset, admin level accounts, add a subject, comment on other people's subjects, and you can also give your subject a location with google maps.

      Technical Stuff

      Build with Model–view–controller architecture has authentication with passportJS using Nodejs, Express, and MongoDB (with Mongoose), Embedded Javascript (EJS), RESTful Routing, Data persistance with a MongoBD Cloud Atlas, also it uses google maps API for Geolocation and it's deployed and hosted on Heroku.

    • Middleman API

      Description

      Middleman is a communication app that let's you create an account an have access to a timeline of people's discussions like a modern suggestion box. it could be implemented by any group of people to listen to what everyone is thinking in that group (the last part is still under construction, right now it's just on pool of discussions).

      Technical Stuff

      A fullstack MERN application build with node, and MongoDB Cloud Atlas for data persistence, the server is an REST API build with express that serves JSON, the client is a Single-page application build with React create-react-app and uses Redux store as a front-end state manager
